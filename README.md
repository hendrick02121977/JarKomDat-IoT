# Jaringan Komunikasi Data dan IoT  
Materi Ajar & Praktikum dengan ESP32 dan Jetson Nano

## 📘 Deskripsi
Repositori ini berisi bahan ajar, praktikum, dan proyek berbasis **ESP32 sensor** dan **Jetson Nano** untuk membangun sistem IoT terpadu. Materi dikembangkan untuk mendukung mata kuliah *Jaringan Komunikasi Data dan IoT* pada program Diploma 4 / Sarjana Terapan Teknik Informatika.

## 🎯 Capaian Pembelajaran
- Menjelaskan konsep jaringan komunikasi data dan protokol IoT.  
- Mengkonfigurasi ESP32 untuk akuisisi data sensor.  
- Mengintegrasikan ESP32 dengan Jetson Nano untuk pemrosesan data.  
- Merancang sistem terpadu IoT berbasis komunikasi data real-time.

## 🚀 Proyek Akhir
Mahasiswa akan membangun sistem IoT terpadu:  
1. Sensor → ESP32 (data akuisisi)  
2. ESP32 → MQTT Broker (komunikasi data)  
3. Jetson Nano → Pemrosesan & analisis data  
4. Dashboard → Visualisasi hasil  

Output berupa prototype sistem yang dapat dipresentasikan dan diuji.
